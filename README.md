# 🚀 Full Stack Web Development Bootcamp

This repository documents my learning journey through **The Complete Web Development Bootcamp** by Angela Yu.

It includes all of my hands-on exercises, coding challenges, mini-projects, capstone projects, and personal notes as I progress from the fundamentals of web development to building complete full-stack applications.

Throughout this bootcamp, I explore modern web technologies, strengthen my problem-solving skills, and gain practical experience by building real-world projects.
